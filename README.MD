# ng2-semantic-ui

Semantic UI Angular 2 Integrations.

## Installation

To install this library, run:

```bash
$ npm install ng2-semantic-ui --save
```

## Development

To generate all `*.js`, `*.js.map` and `*.d.ts` files:

```bash
$ npm run tsc
```
