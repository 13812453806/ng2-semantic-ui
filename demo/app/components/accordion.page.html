Accordion

An accordion allows users to toggle the display of sections of content

Examples

Accordion

Standard accordion

Styled

Adding classes to the root element applies styles

API

<sui-accordion>

Properties

closeOthers 
true
Limits the number of open panels to 1 when true.

<sui-accordion-panel>

Properties

isOpen 
false
Sets whether or not the panel is open.
isDisabled 
false
Sets whether or not the panel is disabled (locks current state).

Events

isOpenChange
Fires whenever the panel is toggled. [(isOpen)] syntax is supported.
